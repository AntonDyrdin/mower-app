# Mower App — Android UI для робота-газонокосилки

## Требования
- Node.js 18+
- Android Studio
- Java 17+

## Быстрый старт

```bash
npm install
npx cap add android
mkdir android/app/src/main/res/xml
cp android-config/network_security_config.xml android/app/src/main/res/xml/
npm run sync
npm run open
```
